# 2. FRDM-MCXN947 Zephyr RTOS 构建到烧录流程

本文记录 FRDM-MCXN947 上从 Zephyr 工程构建、烧录到串口验证的完整流程。  
前提是 Zephyr 环境、Zephyr SDK、LinkServer 已安装完成。

## 0. 快速构建和烧录

为了避免每次手动设置环境变量，可以在 `D:\Zephyr` 下新建脚本：

```powershell
D:\Zephyr\zephyr_env.ps1
```

脚本内容：

```powershell
cd D:\Zephyr
.\zephyrproject\.venv\Scripts\Activate.ps1

cd D:\Zephyr\zephyrproject\zephyr

$env:ZEPHYR_TOOLCHAIN_VARIANT="zephyr"
$env:ZEPHYR_SDK_INSTALL_DIR="D:\Zephyr\sdk"
$env:Path = "C:\NXP\LinkServer_26.3.123;C:\NXP\LinkServer_26.3.123\dist;$env:Path"

Write-Host "Zephyr environment ready." -ForegroundColor Green
Write-Host "Current path: $(Get-Location)"
Write-Host "Board: frdm_mcxn947/mcxn947/cpu0"
```

以后打开 PowerShell 后，只需要执行：

```powershell
D:\Zephyr\zephyr_env.ps1
```

然后构建：

```powershell
west build -p always -b frdm_mcxn947/mcxn947/cpu0 samples\hello_world
```

烧录：

```powershell
west flash
```

如果 PowerShell 不允许运行脚本，先执行一次：

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 1. 启用环境

每次打开新的 PowerShell 后先执行：

```powershell
cd D:\Zephyr
zephyrproject\.venv\Scripts\Activate.ps1
```

进入 Zephyr 工程目录：

```powershell
cd D:\Zephyr\zephyrproject\zephyr
```

设置 Zephyr 工具链环境变量：

```powershell
$env:ZEPHYR_TOOLCHAIN_VARIANT="zephyr"
$env:ZEPHYR_SDK_INSTALL_DIR="D:\Zephyr\sdk"
```

## 2. 构建工程

FRDM-MCXN947 的有效 board target 为：

```powershell
frdm_mcxn947/mcxn947/cpu0
```

先构建 `hello_world`：

```powershell
west build -p always -b frdm_mcxn947/mcxn947/cpu0 samples\hello_world
```

若要构建 `blinky`：

```powershell
west build -p always -b frdm_mcxn947/mcxn947/cpu0 samples\basic\blinky
```

构建成功后会看到类似：

```text
[161/161] Linking C executable zephyr\zephyr.elf
Generating files from ... zephyr.elf for board: frdm_mcxn947/mcxn947/cpu0
```

## 3. LinkServer PATH

`west flash` 需要能找到 `LinkServer.exe`。  
本机安装路径为：

```powershell
C:\NXP\LinkServer_26.3.123
```

如果是新开的 PowerShell，建议先加 PATH：

```powershell
$env:Path = "C:\NXP\LinkServer_26.3.123;C:\NXP\LinkServer_26.3.123\dist;$env:Path"
```

验证：

```powershell
where.exe LinkServer.exe
LinkServer.exe --version
```

## 4. 烧录

直接执行：

```powershell
west flash
```

正常情况下会显示：

```text
west flash: using runner linkserver
INFO: Selected probe ...
INFO: Selected device ...
```

如果出现：

```text
No probes detected
```

说明没有识别到板载调试器，检查：

1. 是否插在 **MCU-Link / Debug USB** 口。
2. USB 线是否支持数据传输。
3. 设备管理器里是否出现 MCU-Link / CMSIS-DAP / VCOM。
4. 驱动弹窗是否已点安装。

## 5. 串口查看

烧录完成后，用串口工具打开板载虚拟串口：

```text
波特率 115200
数据位 8
停止位 1
校验位 None
```

应看到：

```text
*** Booting Zephyr OS build ...
Hello World! frdm_mcxn947/mcxn947/cpu0
```

## 6. 常见问题

### 6.1 找不到 Zephyr SDK

如果构建时提示找不到 `Zephyr-sdk`，确认：

```powershell
$env:ZEPHYR_SDK_INSTALL_DIR="D:\Zephyr\sdk"
```

并且 `D:\Zephyr\sdk` 下确实有 SDK 文件。

### 6.2 找不到 LinkServer

如果 `west flash` 提示：

```text
required program LinkServer.exe not found
```

先确认：

```powershell
where.exe LinkServer.exe
```

必要时重新设置：

```powershell
$env:Path = "C:\NXP\LinkServer_26.3.123;C:\NXP\LinkServer_26.3.123\dist;$env:Path"
```

### 6.3 Configuration Tools 下载失败

`MCUXpresso Configuration Tools` 在当前安装器里下载失败，不影响 Zephyr 构建和烧录。  
本流程只需要：

- Zephyr SDK
- LinkServer
- FRDM-MCXN947 板卡支持
